# tag-picker app

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm start

# build for production with minification
npm run build

# run linter
npm run lint
```

## Check live on

[Tag-picker](https://tag-picker.netlify.com/)
